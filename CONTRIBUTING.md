# Contributing to GEO Auditor

Thanks for your interest.

## How to Contribute

1. **Open an issue first** — bug report or feature request. Let's discuss before you write code.
2. **Fork + branch** — `git checkout -b feature/your-idea`
3. **Keep it small** — one feature or fix per PR.
4. **Test before PR** — run `python3 geo_auditor.py --file test.md` to verify nothing is broken.
5. **Open PR** — link to the issue, describe what changed.

## Code Style

- Python 3.7+ compatible, zero external dependencies.
- The web version (`geo-auditor.html`) and CLI (`geo_auditor.py`) share the same detection logic — changes to one should be reflected in the other.
