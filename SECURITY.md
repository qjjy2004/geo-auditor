# Security Policy

## Reporting a Vulnerability

GEO Auditor is a zero-dependency, offline-first Python tool. No data ever leaves your machine.

If you discover a security vulnerability, please report it privately to:

**Email:** dev@zhibi.xyz

**Do NOT open a public issue.**

We'll acknowledge your report within 48 hours and aim to resolve it within 7 days.

## Scope

- Code-level vulnerabilities in `geo_auditor.py` or `geo-auditor.html`
- Supply chain concerns (though we have zero dependencies)
- Logic flaws that could produce misleading scores

## Out of Scope

- Issues with your specific environment (Python version, OS, encoding)
- Feature requests disguised as security reports
- Third-party services used alongside GEO Auditor (the tool itself is offline)
