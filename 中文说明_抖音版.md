## 文案 A — 痛点直击（适合口播或字幕卡）

**你的文章，AI搜得到吗？**

现在的人不用百度了。
ChatGPT、豆包、元宝、DeepSeek——
张嘴就问，AI直接给答案。

问题是：AI给的答案里，**有你吗？**

我写了几十篇行业文章，拿AI一测，大部分根本不会被引用。
不是写得不好，是写得不像AI会引用的样子。

于是我做了一个工具：**GEO Auditor**。

上传文章 → 14个维度拆解打分 → 告诉你哪里扣分了，怎么改。

不是测文笔好不好，是测**AI搜索引擎愿不愿意引用你**。

跟市面上的工具不一样：
别人是监控——你发了之后告诉你"AI提到你没"
我这是大灯——你发之前就知道"AI会不会引用我"

开源，零依赖，Python 3.7+ 就能跑。
电脑上跑，不联网，文章不会上传到任何服务器。

GitHub 搜：**qjjy2004/geo-auditor**

## 文案 B — 技术流（适合图文/视频讲解）

**我做了一个AI内容检测器，跟你平常用的不一样**

市面上做GEO的工具都是监控：
设好关键词 → 跑AI搜索引擎 → 看AI答案里有没有你的品牌

这叫**后视镜**。告诉你已经发生的事。

我做的这个叫**前大灯**。

你写文章的时候，它能告诉你：
- 这几个地方AI搜索引擎看不懂
- 这段写得太像AI腔，搜索引擎会降权
- 这里加个FAQ模块，引用概率翻倍
- 这个标题关键词密度不够

14个维度逐项打分，满分100。

最狠的是**自我进化**功能：
每次检测 → 自动记录 → 改了再测 → 对比进步 → 积累经验 → 生成写作规则给你AI助手用

你的AI助手写完文章，检测器打分，改了再写，越写越强。

**零依赖。不联网。MIT开源。**
